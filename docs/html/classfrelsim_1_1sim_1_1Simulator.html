void frelsim::sim::Simulator::sim ( )

Run a full simulation and steps until tFinal.

sim() We step to the minimum of the maxStepSize and the next discrete time.

We do this because some components could contain continuous functions which need to be integrated more frequently. This is the intention of the maxStepSize parameter. A discrete event may occur infrequently, so we rely on the user to know how frequently their system needs an integration update to continuous states.
